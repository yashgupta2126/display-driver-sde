# display-driver

This repository contains the display driver for the Qualcomm Linux ecosystem. It provides the essential kernel modules, build files, and configuration necessary to build and run the display driver.

## Features

- Display driver modules for Qualcomm mobile platforms.
- Support for Bazel, Android Build System (Blueprint/Make), and Kbuild build environments.
- Integration with standard Qualcomm Linux display-core services.
